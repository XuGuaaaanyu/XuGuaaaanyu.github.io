Non-linear Least Squares

In general, let r:Rn→Rm,m≥n\mathbf{r}:\mathbb{R}^n\to \mathbb{R}^m,m\geq nr:Rn→Rm,m≥n﻿ be a smooth residual function. We can define an objective function f:Rn→Rf:\mathbb{R}^n\to\mathbb{R}f:Rn→R﻿ as

The goal is still to find the minimizer of the function fff﻿, i.e., x⋆=arg min⁡x∈Rnf(x)\mathbf{x}^{\star}=\argmin_{\mathbf{x}\in\mathbb{R}^n}f(\mathbf{x})x⋆=argminx∈Rn​f(x)﻿. Unfortunately, there’s no closed-form formula for finding x⋆\mathbf{x}^\starx⋆﻿ as we did for the LLS problem. Common methods usually start from an initial guess, and iteratively approximate the optimum based on “descent direction” mentioned before.

Specifically, if we start with an initial guess x0\mathbf{x}^0x0﻿, linearize around x0\mathbf{x}^0x0﻿, and find a descending direction d\mathbf{d}d﻿. Then, we update the guess by moving along direction d\mathbf{d}d﻿ , and we can expect the objective will become smaller, i.e., xk+1=xk+d\mathbf{x}^{k+1 }=\mathbf{x}^k+\mathbf{d}xk+1=xk+d﻿ and f(xk+1)<f(xk)f(\mathbf{x}^{k+1})<f(\mathbf{x}^k)f(xk+1)<f(xk)﻿. Hopefully, this process will converge to a local minimum. Common methods include Gauss-Newton, Levenberg-Marquardt, or gradient descent, distinguished by how they compute d\mathbf{d}d﻿.

def gauss_newton(x0, max_iter, tol):
    x = x0.copy()
    for _ in range(max_iter):
        r = residual(x)
        J = jacobian(x)
        d = np.linalg.solve(J.T @ J, -J.T @ r) # Solve (J^T J) d = -J^T r
        x = x + d
        if np.linalg.norm(d) < tol:
            break
    return x

def gradient_descent(x0, max_iter, tol, eta):
    x = x0.copy()
    for _ in range(max_iter):
        r = residual(x)
        J = jacobian(x)
        d = -eta * J.T @ r # Directly set d to be along the opposite gradient
        x = x + d
        if np.linalg.norm(d) < tol:
            break
    return x

def levenberg_marquardt(x0, max_iter, tol, lam):
    x = x0.copy()
    for _ in range(max_iter):
        r = residual(x)
        J = jacobian(x)
        A = J.T @ J + lam * np.eye(len(x))
        d = np.linalg.solve(A, -J.T @ r) # Solve (J^T J + λI) d = -J^T r
        x = x + d
        if np.linalg.norm(d) < tol:
            break
    return x

Note that the Levenberg-Marquardt method is an interpolation between Gauss-Newton and gradient descent.

  • As λ→0\lambda\to 0λ→0﻿, it reduces to Gauss-Newton: (Jk⊤Jk+λI)→Jk⊤Jk\left(\mathbf{J}_k^\top \mathbf{J}_k+\lambda \mathbf{I}\right) \to \mathbf{J}_k^\top \mathbf{J}_k(Jk⊤​Jk​+λI)→Jk⊤​Jk​﻿.
